2. 设计停车场 有屏幕显示剩余停车位 以及随机生成车位号码，如何设计类以及它们之间的关系？
   * class 停车位  属性 1. id、 2. 状态(满或空)
   * class 停车场  属性 1. 列表 可用停车位  2.列表 已用停车位 3. 屏幕
   * class 屏幕    属性 1. 停车场 方法: 调用停车场的方法去获取剩余车位数和随机车位号
  
   主要是类之间的组合关系，停车场与屏幕之间是组合关系，考察面向对象设计
